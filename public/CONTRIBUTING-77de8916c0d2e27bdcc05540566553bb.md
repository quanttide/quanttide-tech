# 贡献指南

欢迎贡献本项目！

## 贡献方式

- 报告问题：通过 GitHub Issues 反馈
- 提交修改：通过 Pull Request 贡献代码或文档

## 提交流程

1. Fork 仓库
2. 创建分支：`git checkout -b feature/xxx`
3. 进行修改
4. 提交更改：遵循Git提交规范（参见asset/type/git_repo.md）
5. 推送分支：`git push origin branch-name`
6. 创建 Pull Request

## 编写规范

编写文档时遵循：
- 文档格式标准（docs/format.md）
- Git提交规范（asset/type/git_repo.md）

## 发布版本

遵循版本发布标准（devops/release.md）

## 联系方式

如有疑问，可通过 GitHub Issues 联系维护者。
