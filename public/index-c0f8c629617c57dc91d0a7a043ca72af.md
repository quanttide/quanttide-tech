# 量潮数据
# 尝试介绍量潮科研服务到底是什么

今天被一个朋友cue"你的主营业务是什么"，她之前已经cue了我好几遍，但是我每次都没解释明白，十分挫败，所以专门写一篇详细的文章，希望可以把主营业务到底是什么讲明白。因为我们的业务起点主要是量潮科研服务，所以主要介绍这个业务的演化过程，然后顺带着讲其他业务怎么延伸出来的。

## 我们的服务是什么？

设想有个经济学家组成的科研团队正在开启一个关于技术创新的经济学研究课题。他们需要使用来自某个群体的GitHub历史提交数据作为他们参与技术创新的证据。

研究团队有一个通过前期工作积攒下来的GitHub账号名称，并且找到了一个专门提供GitHub的网站。但是这个原始GitHub数据每个压缩包就有几百个G。研究团队希望在一个星期左右就可以获得清洗以后的数据。

我们接到这个需求以后，评估大概需要开发一些Python脚本，并且把程序部署到海外区的云服务器上运行。于是我们根据开发和部署的定价规则给客户一个报价，客户觉得没问题，我们收了首付款就开始执行。一般都可以在交付时间完成。然后这一单就做完了，我们就得到了尾款。

类似地，我们接各类数据采集和数据预处理任务，科研工作者自己完成后续的数据分析和可视化工作。

## 我们怎么获取客户？

我们的客户画像主要是985及以上国内外高校的经管专业科研工作者。他们科研经费相对充足，平均每年有几万到十几万，且需要全部用于科研中；发表压力大，比如说三年需要发表多少多少顶刊。因此为了加快科研产出效率，他们需要把非核心的工作外包给我们做。

我们目前的获客渠道主要有两个支点，厦大校友圈和知乎大V圈。

我在2016年通过公司前身的课题组的早期成果混进了知乎经济学大V圈，他们一直是我和团队的主要顾问和业务支持者。比如，某乎经济学头牌大V司马懿老师四五年来给我们下单过十来次并在持续增加；西交经院樊仲琛老师、袁聪颖老师（我本科毕业以后我本科导师的博士学生）两年来也分别有五六个订单并在持续增加；对外经贸政府管理学院的江汉臣老师、杭州师范大学的冯晗老师等也都有一个或者数个项目。

我们在厦大经济学科的校友圈子里有比较广泛的影响力，年龄跨度约十来届，随着这些校友逐渐开始走上教职岗位，从去年和今年开始，我们的客户群体也在空前扩大。我们在2023年开始系统性地放弃厦大老师的市场，到今年开始完成放弃。由于在自媒体上系统地批评厦大经院获得关注，很多今年已经工作的校友、院友，或者有牛导支持的在读博士生，比如导师是长江学者的浙大博士生沈宇辉博士等找到我们下单。

目前我们产能不足和效率不理想没急着扩新，只要在学术圈持续扩大影响力就很容易获客了，比如我继续写文章批评经济学科的糟糕现状和建议学术研究发展方向，基本上也是写一篇火一篇，一写就忙不过来。

## 如何衍生新业务？

由于我们在大数据、大模型、云计算等方向的积累比较深，我们受到了浙江理工大学计算机系主任沈炜老师和副主任夏劲松老师邀请，今年5月在计算机系为主修学生开设大三专业课《大数据技术》、《大数据项目实践》，我们把我们平时在工作中积累的理论和项目分别带到了两门课上。今年9月开始，我们即将运行教育部审批通过的浙理大数据微专业，为全校同学提供包括《大数据导论》、《数据科学》、《数据工程》、《大数据项目实践》、《生产实习》。这些业务被我们统一归类在"量潮课堂"品牌下。

从事生成式AI开发的企业也类似于经管科研工作者有许多研究性质的数据处理需求，因此我们今年开始也在把客户群体扩大到有数据需求的AI企业及其部门。这些统一以"量潮数据咨询"的品牌对外输出。由于我们的工作中前期沟通和设计占比较大，因此定义为"咨询+实施"的模式比较能够体现我们对市场的预判。
