# 抽取知识卡片

从个人日志中提取已发生、可复用的具体做法的标准化流程。

## 核心步骤
1. **定位操作描述**：通读日志，标记用户描述“做了什么”的语句（行为、步骤、策略、调整动作），忽略抽象框架、理论模型、内容分析。
2. **筛选有效做法**：只保留“已经实际使用”且来源于日志事实描述（现象、反应、结果）的行为，不提取建议或假设。
3. **抽象为具体做法**：用简洁、可执行的语言概括，去除“模型”“战略”“框架”类总结。
4. **命名并结构化**：为每个做法起清晰名称，说明来源（现象/反应），描述下次类似情境的具体操作。

## 验收卡片（质量检查）
每张卡片生成后，按以下标准逐条核对：
- [ ] **来源真实**：做法可直接追溯到日志中的具体语句或事实描述。
- [ ] **已发生**：不是建议或假设，而是用户已经使用过的行为。
- [ ] **可执行**：描述清晰，他人或用户本人下次能照着做。
- [ ] **无冗余**：不含抽象框架、理论模型或内容分析。

若任一项不通过，需返回修改对应卡片。
