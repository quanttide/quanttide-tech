# Specification

格式要点

### 1. 标题格式
- `#` + 空格 + 模型名称（用反引号包围）
- 中文描述紧跟标题

### 2. 字段表格
- 表头：`| 字段 | 类型 | 必填 | 说明 |`
- 类型用反引号：`string`、`enum`、`string (ISO 8601)`
- 必填：是/否
- 说明要简洁

### 3. 枚举说明
- `###` + 枚举名称
- 每个值用加粗：`**past**`
- 冒号后解释用途

### 4. 示例
- `## 示例`
- JSON 代码块，有具体值
- 不是 placeholder，是真实示例

### 5. 整体风格
- 专业、简洁、不废话
- 人类可读
- 代码友好
