# 议题：GitHub数字资产

按照内部租户隔离规范，将 GitHub org 拆分为多个组织，实现公私分离和业务隔离。

## 组织

### quanttide（量潮，public 租户）

量潮主组织，管理量潮第二大脑为中心的开源体系。

### qtarchive（归档站，archive租户）

由原 xmucpp 组织改名，并合并其他旧 org 及主组织中的归档仓库。

### qtadmin（量潮管理后台，qtadmin租户）

量潮管理后台相关仓库。将主组织中以 qtadmin 开头命名或实际属于管理后台业务的仓库迁移至此，统一重命名仓库前缀。

### qtdata（量潮数据，qtdata租户）

量潮数据业务相关仓库。将主组织中以 qtdata 开头命名或实际属于数据业务的仓库迁移至此，统一重命名仓库前缀。

### qtclass（量潮课堂，qtclass租户）

量潮课堂业务相关仓库。将主组织中以 qtclass 开头命名或实际属于课堂业务的仓库迁移至此，统一重命名仓库前缀。
