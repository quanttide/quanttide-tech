# garment-factory-cleaner 文档

## 业务背景

某制衣厂需要分析生产效率，但数据分散在多个 Excel 文件中：
- 产量数据（谁、哪天、做了什么工序、完成多少件）
- 工序表（每个款式的标准工序、机器、时间）
- 返工记录（哪些产品需要返工）
- 考勤记录（员工出勤时间）

本项目将 4 个数据源按款号、日期、员工维度合并，生成统一的分析数据集，用于：
- 员工绩效计算
- 款式生产效率分析
- 返工率影响评估
- 成本核算

## 业务知识

### 产值公式

制造业产值由三要素构成：

```
产值 = 员工 × 工序 × 产量
```

| 要素 | 问题 | 关键数据 |
|------|------|----------|
| 员工 | 谁做的 | 姓名、出勤时间 |
| 工序 | 做了什么 | 工序名称、标准时间、机器类型 |
| 产量 | 做了多少 | 产量数量、返工数量 |

### 工序

工序是完成一件服装所需的**具体生产步骤**。一条裤子可能包含：上腰头、埋夹、装拉链、压明线、订纽扣等工序。

每个工序有标准属性：
- **标准时间**：完成该工序的理论用时（秒），用于计算效率
- **机器类型**：所需设备（平车、埋夹机、拷边机等）
- **工序等级**：难度等级，影响计件单价
- **部位**：所属部位（腰头、领子、袖子等）

### 款号

款号是款式编号，如 `12C1450`、`16A0509`。同一款号对应一套固定的工序组合。

原始数据中款号格式混乱，需要清洗：
- 去除后缀：`16A0509-12-28` → `16A0509`
- 修正拼写：`15CO156` → `15C0156`
- 同款映射：`67C0015` → `12C1470`

### 返工

返工是指已完成的产品因质量问题需要重新加工。返工数量的业务影响：

- **有效产量**：实际有效产量 = 产量 - 返工数量
- **效率计算**：返工消耗额外时间，降低实际效率
- **质量评估**：返工率反映员工技能水平或工序难度
- **成本核算**：返工增加人工成本，需计入总成本

返工率公式：`返工率 = 返工数量 / 产量 × 100%`
