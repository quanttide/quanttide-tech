# 量潮科技议事章程

## 一、两院制概述

两院制（Bicameralism）模型来源于罗马共和国的元老院-公民大会制度。

### 对应关系

| 现代名称 | 罗马名称 | 职能 |
|----------|----------|------|
| 合伙人委员会 | 元老院 | 所有权、决策权 |
| 书记处 | 执政官 | 执行协调 |
| 公司代表大会 | 公民大会 | 经营权、民意表达 |

### 设计思想

- 所有权和经营权分离
- 合伙人掌握立宪权和司法权
- 其他事务交由代表大会处理

## 二、组织架构

```
合伙人委员会
    └── 书记处
            └── 公司代表大会（周会）
                    ├── 执行委员会（管理会）
                    └── 技术委员会（技术会）
```

| 机构 | 会议名称 | 频率 |
|------|----------|------|
| 合伙人委员会 | - | 定期 |
| 书记处 | 办公会 | 每周一次 |
| 公司代表大会 | 周会 | 每周一次 |
| 执行委员会 | 管理会 | 每周一次 |
| 技术委员会 | 技术会 | 每周一次 |

## 三、职责划分

### 合伙人委员会

类似一般公司的董事会职能，暂无外部董事。

- 掌握立宪权：制定制度
- 掌握司法权：审议执行
- 审核书记处提交的重大事项

### 书记处

- 由轮值书记负责统筹
- 协调领导各个部门
- 审议各会议内容，了解全局
- 提前整理材料提交至代表大会

### 公司代表大会

- 决策方式：需全体与会人员明确表决通过
- 材料来源：书记处汇总整理各委员会提案

### 执行委员会

- 由秘书长主持，部门负责人参与
- 每周精选 1-2 件核心管理事项，深入分析原因
- 形成提案后提交至代表大会

### 技术委员会

- 由 CTO助理主持，技术负责人参与
- 讨论技术事务，形成提案

## 四、运作流程

```
发现问题 → 对应委员会会议讨论 → 形成提案 → 书记处汇总整理 → 代表大会决策 → 循环改进
```

## 五、新人培养机制

- 轮值书记参与书记处会议
- 优先安排有培养空间的人员
- SSC 成员侧重参与书记处工作，借此了解全局
