# 招聘考核

定义招聘过程中的考核标准设置原则。

## 考核原则

### 聚焦实际能力

考核题目应聚焦岗位核心能力，避免形式化官僚考察。

以新媒体运营为例，核心能力是选题和写作。考察重点是能否从原始内容中找到好的叙事角度，并邀请相关人员从多视角讲述故事。

### 透明化标准

考核标准应在招聘前明确：
- 提前告知候选人考察方向
- 试题与岗位实际工作高度相关
- 评分标准公开可解释

## 笔试题设计

### 设计要点

1. 来源于实际工作：从岗位实际任务中提炼考核点
2. 考察解决问题的能力：而非知识记忆
3. 提供必要素材：如写作素材、案例材料等
4. 结果导向：要求候选人提出有效方案，而非仅批评现状。结果导向是ISO质量管理体系与绩效管理理论的核心思想，强调以实际成果作为衡量标准。以新媒体运营为例，重点考察能否提出有效的运营策略。

### 示例

新媒体运营笔试题：提供若干原始实录内容，要求候选人从中选择可写作题目，并说明选择理由和写作角度。重点考察能否提出可执行的解决方案。

## 面试配合

笔试题结果作为面试讨论素材，深入考察候选人思路和潜力。

### 模拟会议讨论

面试模拟内部会议环境，围绕笔试提案深入讨论。

考察重点：
- 协作意识：能否听取他人意见、整合多方观点
- 反应能力：面对质疑时的应对策略
- 表达说服：清晰阐述观点并影响他人
- 结果导向：能否提出有效的运营策略，而非只会否定现状

淘汰标准：固执己见、无法接受反馈的人。

### 软能力评估

- 能否在讨论中调整自己观点
- 能否与他人有效沟通
- 能否承受压力和质疑
