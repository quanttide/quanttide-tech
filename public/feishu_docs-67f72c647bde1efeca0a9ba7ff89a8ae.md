# 飞书文档

## 素材1

知识库

在我越来越少介入一线以后，知识库是目前最常用的功能。

手机端 App 的知识库功能只做了一半。比如，手机端是不能创建知识库的，也不能调整知识库设置。

有时候手机上需要创建一个新知识库整理旧知识库的内容，或者给缺少权限的同事授权，还得跑去电脑端操作，非常耽误时间。

手机端登录网页版，并且调整到桌面模式可以解决这个问题。特别注意，手机网页和手机 App 是一样的接口。如果在电脑网页创建知识库以后，自动跳转到手机页面会打不开，要再手动点击“请求桌面网页”。

其他问题多了去了，下次再吐槽。

文档

编辑器各种奇奇怪怪的问题非常多。吞文字啊丢文字啊乱跑啊，在手机端上问题也非常多。

电脑端的问题是，有时候在 App 内部，有时候跳转到网页。真不知道这个草台班子逻辑是怎么搞出来的。

吹的天花乱坠的飞书文档，稳定性和腾讯文档也就是半斤八两吧。写个文档还不够耽误时间的。
