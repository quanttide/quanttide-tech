# 学习日志手册

## 1. 概述
本手册基于《工作论文：日志规范原理》，专门针对学习日志的记录提供指导。

## 2. 学习日志的特殊性
学习日志作为主体（第二大脑）的自我认知记录，具有以下特点：
- **主体性**：记录者是第二大脑自身
- **反思性**：包含对学习过程的反思
- **行动导向**：连接学习与后续行动

## 3. 记录模板
### 3.1 标题
格式：`YYYY-MM-DD 学习 [主题]`

### 3.2 内容结构
- **学习目标**：明确本次学习的目的
- **学习内容**：详细的观察、记录、框架理解
- **学习收获**：总结关键洞察
- **下一步行动**：基于学习的行动计划

## 4. 存储规范
- **位置**：`meta/journal/learn/`
- **命名**：`YYYY-MM-DD.md`

## 5. 规范参考
- 通用日志规范：`meta/journal/README.md`
- 学习日志章程：`meta/journal/learn/README.md`
