# 学习日志章程

## 定义
学习日志是 **陈述型记忆** 在 **现在** 时间轴、**事件类** 分类下的原始记录。
它是主体的学习记录。meta 模块的主体是第二大脑自己。

## 1. 概述
本章程基于《工作论文：日志规范原理》，专门针对学习日志的记录提供指导。

## 2. 学习日志的特殊性
学习日志作为主体（第二大脑）的自我认知记录，具有以下特点：
- **主体性**：记录者是第二大脑自身
- **反思性**：包含对学习过程的反思
- **行动导向**：连接学习与后续行动

## 3. 结构
- **标题**: `YYYY-MM-DD 学习 [主题]`
- **学习目标**: 明确本次学习的目的
- **学习内容**: 详细的观察、记录、框架理解
- **学习收获**: 总结关键洞察
- **下一步行动**: 基于学习的行动计划

## 4. 文件命名规范
- **位置**: `meta/journal/learn/`
- **格式**: `YYYY-MM-DD.md` (例如: `2026-03-18.md`)
- **说明**: 学习日志按日期命名，方便追踪和查找

参考 `meta/journal/learn/2026-03-18.md`。
